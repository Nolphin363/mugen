仮面ライダーカブト  	070728版	2007/07/28
出典「仮面ライダーカブト」

仮面ライダーカブトをダウンロードしていただき、ありがとうございます。

本作完結から相当遅れての公開となってしまいましたが、MUGEN版もお届けいたします。

カブトは、多彩な連続打撃で相手を後退させて追い込んでいくような戦い方をすることが多かった印象があるので、派生型の連続技をつけました。
現状は基本ともいえる2択の分岐だけですが、これに下段や対空などの、もっと多彩な分岐を追加して、相手を揺さぶるような闘い方をするキャラクターにしていきたいと思っています。
カブトは、個々の技性能が高い分、技の数自体は少ないという、ガイルのようなキャラにしたいと思って製作しているので、技の数自体を増やす事は考えていません。
本来ならクナイガンをつかった飛び道具なんかがあるべきかとも思いますが、他のライダーとの差別化を考えて、あえて近接戦型のキャラにしていきたいと思います。


  コントロール（動き）
  --------

    Basic Controls
    --------------
　　　仮面ライダーカブトは3ボタンのキャラです。
      
      D - 下
      F - 前
      U - 上
      B - 後
      x - NO USED
      y - NO USED
      z - NO USED
      a - パンチ
      b - キック
　　　c - クナイガン

    通常の動き
    -----------------
      U              - ジャンプ
      UF             - 前ジャンプ
      F              - 前移動
      DF             - 屈み
      D              - 屈み
      DB             - 屈みかガード
      B              - 後移動かガード
      UB             - 後ジャンプ
      ab             - リカバー 

    投げ
    ------
      F/B + c	　　　- オラオラ

    必殺技
    ---------------
　　　D, DF, F + a   　　	- アバランチラッシュ：追加入力型の連撃技
　　　追加入力　a a a b		- ルートA：最大5連撃
　　　追加入力　a b		- ルートB：3連撃最後が中段攻撃

　　　D, DF, F + c   　　	- アバランチスラッシュ：追加入力型の連撃技　派生無し
　　　追加入力　c c

    　D, DB, B + a/c   　　	- アッパースラッシュ：対空技

　　Level.1 超必殺技
    -------------      
      D, DF, F ×2 + c		- ライダーキック

　　Level.3 超必殺技
    -------------      
      a, a, F, b, c		- クロックアップ：一定時間時間停止
